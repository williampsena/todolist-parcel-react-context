## Requirements

 - Node.js (v.10.x)
 - Parcel.js
 - Yarn (optional)

## Setup

First of all install external dependencies.

```
# bundler
npm install -g parcel-bundler

# project packages
npm install
```

Then install local dependencies.

```
# npm
npm install

# yarn
yarn install
```

## Run

```
npm start
```

## Build

```
npm run build
```
